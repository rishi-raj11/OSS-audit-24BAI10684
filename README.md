## How to Run Scripts in VS Code

1. Open **VS Code**.  
2. Open the project folder in VS Code:
   - Click **File → Open Folder**  
   - Select the folder containing all scripts (`script1_system_info.py`, etc.)

3. Open **Terminal in VS Code**:
   - Click **Terminal → New Terminal**  

4. Make sure Python 3 is installed (check version):
 bash python3 --version

5. Run each script individually in the terminal:
 python3 script1_system_info.py
 python3 script2_disk_usage.py
 python3 script3_users_groups.py
 python3 script4_packages.py
 python3 script5_lamp_test.py
